⚙️ Dashboard Setup

The Predicts Hub dashboard reads live data relayed through Slack every 25 minutes by a Cowork scheduled task. You need a Slack User OAuth Token with search:read permission to connect.

1 Go to api.slack.com/apps → Create New App → From scratch
2 Name it Predicts Dashboard, select your FanDuel workspace, then click Create App
3 Go to OAuth & Permissions → scroll to User Token Scopes → Add search:read
4 Scroll up and click Install to Workspace, authorize it, then copy the User OAuth Token (starts with xoxp-)
